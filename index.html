Cosa sono i Microsoft Cognitive Services?

Gli 8 processi cognitivi principali sono:

  • Percezione
  • Apprendimento
  • Linguaggio
  • Pensiero
  • Attenzione
  • Memoria
  • Motivazione
  • Emozione

Cosa sono i Microsoft Cognitive Services?

Perché non fare in modo che anche le applicazioni non abbiano la possibilità di avere delle capacità cognitive che permettano loro di:

  • Vedere
  • Comprendere il linguaggio
  • Prendere delle decisioni

Cosa sono i Microsoft Cognitive Services?

Microsoft con i sui “cognitive services” fornisce agli sviluppatori un insieme di servizi Azure come API, SDK e per aiutare gli sviluppatori a creare applicazioni intelligenti senza disporre di competenze o conoscenze di data science e profonde conoscenze di machine learning.

Vision Service: Computer Vision

Comprende un set di API che consentono l'elaborazione avanzata di immagini e l'estrazioni di metadati ed informazioni dalle stesse.

Ad esempio permettono di:

  1. Riconoscere oggetti contenuti in immagini.
  2. Estrarre blocchi di testo da immagini.
  3. Rilevare volti in un immagine.
  4. Classificare/taggare un immagine in base al contenuto in modo automatico.

Language Services

Comprende un set di API che consentone l'elaborazione del linguaggio naturale al fine di interpretare sentimenti ed intenti dell'utente.

Ad esempio permettono di:

  1. Analisi, comprensione ed interpretazione di testo (LUIS)
  2. Traduzione di testo.
  3. Conversazione Bot Style
