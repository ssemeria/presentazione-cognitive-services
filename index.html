Cosa sono i Microsoft Cognitive Services?

Microsoft con i sui “cognitive services” fornisce agli sviluppatori un insieme di servizi Azure sotto forma di REST API ed SDK per aiutare gli sviluppatori a creare applicazioni intelligenti senza disporre di competenze o conoscenze di data science e profonde conoscenze di machine learning.

Cosa sono i Microsoft Cognitive Services?

Servizi cognitivi di Azure consentono agli sviluppatori di aggiungere facilmente funzionalità cognitive nelle applicazioni. L'obiettivo di Servizi cognitivi di Azure è aiutare gli sviluppatori a creare applicazioni in grado di visualizzare, ascoltare, parlare, comprendere e persino iniziare a ragionare.

Setup e Definizioni Preliminari

Il numero dei Cognitive Service offerti continua ad aumentare.
Perdersi diventa molto facile!

Diamo alcune definizioni:

  • Servizio: un raggruppamento logico di funzionalità affini. Si compone di API
  • API: un insieme di risorse remote (nel senso REST del termine) che possiamo chiamare Endpoint
  • Endpoint: singole risorse cui possiamo accedere mediante API REST

Vision Service

Comprende un set di API che consentono l'elaborazione avanzata di immagini e l'estrazioni di metadati ed informazioni dalle stesse.

Ad esempio permettono di:

  1. Riconoscere oggetti contenuti in immagini.
  2. Estrarre blocchi di testo da immagini.
  3. Rilevare volti in un immagine.
  4. Classificare/taggare un immagine in base al contenuto in modo automatico.

Speech Services

Set di API che consentono la conversione del parlato in testo e di stringhe di testo in parlato "naturale".

Traduzione del parlato in altre lingue e riconoscimento del soggetto parlante.

Ad esempio:

  1. Trascrivere una conversazione
  2. Traduzione simultanea
  3. Identificazione del parlante
  4. Analisi del parlato e riconoscimento di intenti

Decision Services

Set di API che forniscono raccomandazioni e suggerimenti o che aiutano in procedure di classificazione automatica.

Ad esempio:

  1. Rilevare anomalie in set di dati, sulla base del pregresso
  2. Tagging automatico di contenuti testuali, audio e video per rilevare contenuti offensivi, pericolosi o non desiderati
  3. Suggerimento di contenuti da presentare ad un cliente sulla base dei suoi gusti ("potrebbe piacerti anche questo...")
